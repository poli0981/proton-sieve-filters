# 📚 REFERENCES AND RESOURCES

This document contains comprehensive references, documentation sources, technical specifications, and additional resources used in the development and maintenance of the Proton Sieve Filters project.

## 📧 Email Filtering and Sieve Standard Documentation

### Core Sieve Specifications
**RFC 5228: Sieve - A Mail Filtering Language**
- [https://tools.ietf.org/html/rfc5228](https://tools.ietf.org/html/rfc5228)
- Official Internet Engineering Task Force (IETF) specification
- Foundation document for Sieve scripting language
- Syntax, semantics, and base functionality definitions

**RFC 5229: Sieve Email Filtering - Variables Extension**
- [https://tools.ietf.org/html/rfc5229](https://tools.ietf.org/html/rfc5229)
- Variable handling and manipulation in Sieve scripts
- Advanced scripting capabilities and conditional logic

**RFC 5230: Sieve Email Filtering - Vacation Extension**
- [https://tools.ietf.org/html/rfc5230](https://tools.ietf.org/html/rfc5230)
- Auto-reply and vacation message functionality
- Out-of-office message handling best practices

**RFC 5231: Sieve Email Filtering - Relational Extension**
- [https://tools.ietf.org/html/rfc5231](https://tools.ietf.org/html/rfc5231)
- Comparative operations and advanced matching capabilities
- Numerical and string comparison functions

**RFC 5232: Sieve Email Filtering - Imap4flags Extension**
- [https://tools.ietf.org/html/rfc5232](https://tools.ietf.org/html/rfc5232)
- IMAP flag manipulation and email state management
- Message marking and categorization functions

**RFC 5233: Sieve Email Filtering - Subaddress Extension**
- [https://tools.ietf.org/html/rfc5233](https://tools.ietf.org/html/rfc5233)
- Email address parsing and subaddress handling
- Plus addressing and email routing capabilities
