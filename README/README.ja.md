## 📧 説明

このプロジェクトは、Proton Mailユーザー向けに包括的なSieveメールフィルタリングスクリプトを提供します。Sieveはサーバーサイドメールフィルタリング用のドメイン固有言語(DSL)で、メールの分類、既読マーク、有効期限設定、受信箱の効率的な整理などの自動化されたアクションを可能にします。

## ⚠️ 重要なセットアップ要件

**🚨 重要**: これらのフィルターを使用する前に、Proton Mailアカウントでターゲットフォルダを作成する必要があります。**フォルダが存在しないとスクリプトは動作しません。**

### 作成が必要なフォルダ:
```
📁 Shopping      (ショッピング)
📁 Travel        (旅行)  
📁 Work          (仕事)
📁 Security      (セキュリティ)
📁 Entertainment (エンターテインメント)
📁 Gaming        (ゲーム)
📁 Social        (ソーシャル)
📁 Invoices      (請求書)
📁 Newsletters   (ニュースレター)
📁 Health        (健康)
📁 Study         (学習)
📁 Proton        (Protonサービス)
📁 Spam_Filter   (スパムフィルター)
📁 EULA          (利用規約)
```

**Proton Mailでフォルダを作成する方法:**
1. **設定** → **フォルダとラベル** に移動
2. **フォルダを追加** をクリック
3. 上記にリストされた各フォルダを作成
4. **その後で** Sieveフィルターをインストール

## 🚀 主な機能

- **🎯 15+の専門フィルター**: デジタルライフのあらゆる側面を包括するメール分類
- **📱 簡単インストール**: スクリプトをProton Mailに直接コピー&ペースト
- **⚙️ 完全カスタマイズ可能**: ドメイン、キーワード、保持期間の編集
- **🌍 多言語サポート**: 英語、ベトナム語、中国語、日本語のキーワード
- **⏰ スマート保持**: 重要性に基づく自動メール期限切れ
- **🔒 セキュリティ重視**: 強化されたスパム検出とセキュリティアラート優先順位

## 📥 インストールガイド

### ステップ1: 必要なフォルダを作成
1. **Proton Mail** アカウントにログイン
2. **設定** → **フォルダとラベル** に移動
3. 上記の「作成が必要なフォルダ」セクションにリストされたすべてのフォルダを作成

### ステップ2: フィルターをインストール
1. **設定** → **フィルター** → **Sieveフィルターを追加** に移動
2. `filter/`ディレクトリからフィルターを選択 (例: `shopping.sieve`)
3. スクリプト全体をコピー
4. Proton MailのSieveフィルターエディタにペースト
5. わかりやすい名前を付ける (例: "ショッピングフィルター v0.2.0")
6. 完全に有効化する前に **いくつかのメールでテスト**
7. 保存してフィルターを有効化

### ステップ3: カスタマイズ (オプション)
- スクリプトファイル内のドメインリストを編集
- 言語/地域に合わせてキーワードを修正
- ニーズに基づいて保持期間を調整
- 個人のメールアドレスをホワイトリストに追加

## 🧪 フィルターのテスト

### 本格運用前:
1. **小規模から開始**: 一度に一つのフィルターを有効化
2. **アクティビティを監視**: 24-48時間フィルターログをチェック
3. **フォルダを確認**: メールが正しく分類されていることを確認
4. **偽陽性をチェック**: 最初は毎日フィルターされたメールを確認
5. **ルールを調整**: 必要に応じてドメインとキーワードを微調整

## 🚨 トラブルシューティング

### よくある問題:

**フィルターが動作しない**
- ✅ すべてのターゲットフォルダが存在することを確認
- ✅ Sieve構文エラーをチェック
- ✅ フィルターが有効化されていることを確認
- ✅ サンプルメールでテスト

**重要なメールがフィルターされる**
- ✅ 送信者を個人アドレス帳に追加 (ホワイトリスト)
- ✅ ドメイン/キーワードリストを確認・調整
- ✅ フィルター順序の優先度をチェック
