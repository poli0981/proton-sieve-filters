# Proton Sieve Filters

# 説明
このリポジトリは、Proton Mail ユーザーのためにすぐに使える Sieve スクリプトを提供します。Sieve はサーバー側電子メールフィルタリングのためのドメイン固有言語 (DSL) で、電子メールの移動、既読マーク、または期限切れなどのアクションを許可します。ソーシャルメディアフィルターは起点で、他のカテゴリの拡張があります。

**ドメインに関する注意**: 異なるフィルター (例: ソーシャルメディア、請求書) のドメインリストは 'domains.txt' に保存されます。このファイルを編集してドメインを追加または削除できます。スクリプトを変更せずに。

## 制限
- これらのスクリプトは Proton Mail 用に最適化されており、他のメールサービスでは修正が必要になる場合があります (例: "vnd.proton.expire" などの Proton 固有の拡張を削除)。
- カスタマイズには Sieve (または類似のスクリプト) の基本的なプログラミング知識をおすすめします。エラーはメールの見逃しや拒否を引き起こす可能性があります。
- フィルターは受信メールにのみ適用されます。既存のメッセージを再整理しません。
- 潜在的なリスク: ルールが広すぎると偽陽性 (例: 重要なメールの拒否) を引き起こす可能性があります。常に徹底的にテストしてください。
- 保証なし: オープンソースの基準に従い、自己責任で使用してください。

## 使用方法
1. Proton Mail にログイン → 設定 → フィルター → Sieve フィルターを追加。
2. .sieve ファイル（例: filters/social_media.sieve）からスクリプトをコピーして貼り付け。
3. 必要に応じて 'domains.txt' のドメインをカスタマイズ。
4. サンプルメールでテスト。

**利用可能なフィルター**：
- ソーシャルメディア: X、Facebook などの通知をフィルタリング（リポジトリの最初のフィルター）。
- 請求書: PayPal、Amazon などの請求メールを移動。
- 広告: プロモーションメールを処理。
- さらに追加予定...
