## 描述
此仓库提供 Proton Mail 用户的即用 Sieve 脚本。Sieve 是一种特定领域语言 (DSL)，用于服务器端电子邮件过滤，允许移动电子邮件、标记为已读或过期等操作。社交媒体过滤器是起点，并扩展到其他类别。

**域名注意事项**：不同过滤器的域名列表（例如，社交媒体、发票）存储在 'domains.txt' 中。您可以编辑此文件来添加或删除域名，而无需修改脚本。

## 限制
- 这些脚本针对 Proton Mail 优化，可能需要修改以用于其他电子邮件服务（例如，删除 Proton 特定的扩展如 "vnd.proton.expire"）。
- 推荐具有 Sieve（或类似脚本）的基本编程知识进行自定义，因为错误可能导致邮件丢失或拒绝。
- 过滤器仅适用于传入邮件；它们不会重新组织现有消息。
- 潜在风险：规则过于宽泛可能导致假阳性（例如，拒绝重要邮件）；始终彻底测试。
- 无保修：使用风险自负，按照开源标准。

## 使用
1. 登录 Proton Mail → 设置 → 过滤器 → 添加 Sieve 过滤器。
2. 从 .sieve 文件（例如，filters/social_media.sieve）复制脚本并粘贴。
3. 如需必要，在 'domains.txt' 中自定义域。
4. 使用样本电子邮件测试。

**可用过滤器**：
- 社交媒体：过滤来自 X、Facebook 等通知（存储库中的第一个过滤器）。
- 发票：移动来自 PayPal、Amazon 等计费电子邮件。
- 广告：处理促销电子邮件。
- 更多即将到来...
