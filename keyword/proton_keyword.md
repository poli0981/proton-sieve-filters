# Proton Keywords for Email Filtering

> **📝 Important Notes:**
> - These keywords are relative and can be customized to fit your preferences and language
> - You can adapt these keywords for different languages (English, Vietnamese, Chinese, Japanese)
> - Keywords are organized by categories for easy management and customization
> - Retention periods are suggestions and can be adjusted based on your needs

## 📋 Categories Overview

| Category | Retention Period | Description |
|----------|------------------|-------------|
| **Critical Security** | 90 days | Security breaches, unauthorized access |
| **Billing & Finance** | 365 days | Payment records, subscriptions |
| **Account Management** | 30 days | Profile changes, settings |
| **Daily Notifications** | 1 day | Inbox alerts, message counts |
| **Product Updates** | 14 days | Feature announcements, newsletters |
| **Service Status** | 7 days | Maintenance, system updates |
| **Legal Updates** | 90 days | Policy changes, compliance |

## ⚙️ Customization Tips

### Priority Levels
- **Critical (Unflagged + Flagged)**: Security alerts requiring immediate attention
- **Important (365 days)**: Financial records for tax/legal purposes
- **Moderate (30 days)**: Account changes and settings
- **Low (1-7 days)**: Daily notifications and temporary updates

### Size-based Filtering
- **Large files (1M+)**: Billing statements with attachments
- **Medium files (500K)**: Product updates with images
- **Small files (200K)**: Daily notifications and alerts

### Exclusion Patterns
- Exclude emails marked as "important", "urgent", "critical" from quick deletion
- Preserve emails with attachments longer than plain text notifications
